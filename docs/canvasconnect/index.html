CanvasConnect

A zero-dependency Python 3.11+ CLI that bridges HGrader exports to Canvas LMS. Inspect packages, load rosters, fuzzy-match students, post grades, and upload PDFs — all through the Canvas API.

Capabilities

Built for the last mile into Canvas

Everything you need to move HGrader output into the gradebook safely and repeatably.

◇

Export Inspection

Walks HGrader export folders, validates structure, and assembles per-student PDFs from scans and metadata.

👥

Roster Loading

Pulls your course roster from the Canvas API and merges optional Gradebook CSV exports for full coverage.

≈

Fuzzy Matching

Maps local student names to Canvas users with sequence-based scoring, thresholds, and a review step for edge cases.

📊

Grade Posting

Posts scores through the Canvas REST API with support for per-submission updates and bulk operations.

📄

PDF Upload

Uploads assembled PDFs as file submissions so graded work shows up where instructors already review — SpeedGrader.

🛡

Safety Controls

Guards around assignment state, manual posting, and submission locking so you do not accidentally publish too early.

CLI

Commands

Run the full pipeline with run, or execute each stage on its own for debugging and dry runs.

CommandWhat it does
inspect-exportParse export folders, validate contents, and build submission PDFs.
load-rosterFetch the course roster from Canvas and/or a Gradebook CSV export.
matchMatch local students to Canvas roster rows using fuzzy scoring.
review-matchesInteractively confirm or fix ambiguous matches before posting.
build-grade-csvEmit a Canvas-compatible gradebook import CSV when you want a file-based path.
post-gradesPush scores to Canvas submissions via the API.
upload-submissionsUpload each student PDF as an online upload submission.
runChain the stages end-to-end from inspection through upload.
Prerequisites

Requirements

CanvasConnect is stdlib-only — no pip install step. You bring credentials, exports, and a target assignment.

3.11

Python 3.11+

Uses modern stdlib features (including tomllib). No third-party packages required.

🔑

Canvas API token

A personal or integration token with permission to read the course roster and write grades and submissions.

📦

HGrader export folders

Unpacked session exports from HGrader — the same JSON, CSV, and image bundle the app produces.

📝

Canvas assignment

A course assignment configured for file uploads, aligned with how PDFs will be attached for students.

Onboarding

Quick start

Three steps: configure paths and IDs, export your token, then run the pipeline.

1. Configure

Copy the example config, then set your Canvas base URL, course ID, assignment ID, and export paths.

cp config.example.toml config.toml
# Edit config.toml — canvas_url, course_id, assignment_id, export paths

2. Set token

Export CANVAS_API_TOKEN in the shell where you invoke the CLI (or use your platform’s secret store and inline it for CI).

export CANVAS_API_TOKEN="your-token-here"

3. Run

Execute the bundled entrypoint to walk the full bridge from export folder to Canvas.

./canvas-connect run
Step-by-step mode
Prefer to inspect each stage? Run inspect-export, load-roster, match, and the rest individually — see the User Guide for flags and examples.
